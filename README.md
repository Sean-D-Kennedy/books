# books

This Spring Boot project has been updated to run on **Java 25**.

## What changed
- Spring Boot parent upgraded to **3.5.5**
- `java.version` set to **25**

## IDE note (IntelliJ)
The `pom.xml` changes are in Git, but IDE settings are local.  
If you open this project in IntelliJ, make sure:
- **Project SDK** is set to **JDK 25**
- **Maven importer / runner JDK** is set to **JDK 25** (or “Use Project SDK”)
